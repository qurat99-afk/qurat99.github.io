Projects

Below, I’m excited to share a selection of projects I’ve worked on. Each one reflects my growing skills, creativity, and passion for coding—hope you enjoy exploring them as much as I enjoyed building them!

Dice Game with Python

A simple interactive game where players roll dice and compete to reach a target score. It features turn-based gameplay that cycles through multiple players automatically. The program updates and displays scores after each round to keep the competition engaging. Basic control logic ensures fair play and determines the winner efficiently.

Simple Python Calculator

A simple Python calculator is a basic program that performs arithmetic operations such as addition, subtraction, multiplication, and division. It takes input from the user, processes the selected operation using simple logic, and displays the result. This type of program is useful for beginners to understand fundamental programming concepts like input, output, and conditional statements.

Number Guessing Game with Java

A number guessing game is a simple Java game where the program generates a random number and the player tries to guess it. The program gives hints like “too high” or “too low” after each attempt. It helps beginners learn loops, conditions, and user input handling.

Tic Tac Toe Java

Tic-Tac-Toe is a basic two-player game played on a 3×3 grid where players take turns marking X and O. The goal is to get three symbols in a row, column, or diagonal to win. This project helps in understanding arrays, game logic, and control structures.

Hospital Class Diagram

This project showcases a hospital management system through a structured class diagram designed in draw.io. It models key components such as patients, doctors, appointments, and records, along with their relationships. The diagram demonstrates clear understanding of object-oriented design and system architecture.
